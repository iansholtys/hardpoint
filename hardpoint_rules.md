## Characters

Every character has 3 primary stats: **Mind**, **Body** and **Spirit**. Each primary stat acts as a modifier for skill checks and derived stats.

### Skills

Skills represent a character's aptitude for certain actions. Skills act as modifiers for skill checks and some derived stats.

#### General skills

General skills are typically used outside vehicles. Each has 1 or more primary stats which can add their modifier to the roll. If several primary stats are available, the player must choose one, and should explain how they're using that stat to perform the skill.

#### Pilot skills

Pilot skills are typically used inside vehicles. Pilot skills add the modifiers for all of the character's primary stats.

### Occupations

Every character has an occupation, representing their background. Occupations provide skill points to one or more skills (typically general or social skills, but may include personal combat skills).

## Character creation

1. Set primary stats. Each primary stat must be between -1 and +2, and they must all total to 1.
2. Select an occupation and add its associated skill points.
3. For pilots, set each Pilot skill to 1.
4. Spend skill points on skills. No skill may exceed 2 points during character creation.
  1. Allocate 10 additional points to General skills.
  2. For pilots, allocate 2 additional points to Pilot skills.

*Note*: A GM may adjust the game difficulty by changing the starting primary stat total or number of additional General or Pilot skill points.
