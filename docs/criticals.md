# Criticals

## Critical Success
Requirements:
- Dice roll must be at least (3) or above the target difficulty (or defense during attacks)
- Hard mode would also require a max # on the die

Effect:
- For an attack, improve it with 2AP (Armor Piercing)
- For an attack, ability to directly target a Hard Point (for example, target an engine or shield generator)
- For skill checks, GM creates a minor situational bonus related to the Critical Success. This could manifest as a bonus to initiative for the next round, reduced difficulty of the next skill check of that type, etc.
- Add 1 XP for the skill that succeeded

## Critical Failure
Requirements:
- Dice roll must be at least (-3) or below the target difficulty (or defense during attacks)
- Dice roll must have been a 1
- Players must opt-in by asking the GM for a Critical Failure
- Character has not suffered a Critical Failure in this scene

Effect:
- GM creates a hardship related to the Critical Failure. This could manifest as a gun jam, power flickering, engine failure, etc. A Critical Failure should not be lethal
- Add 1 XP for the skill that failed.
